# 💫 About Me

Student Research Assistant at the University of Münster, building open-science tooling for the [FORRT](https://forrt.org) initiative — a TypeScript/Manifest V3 Chromium extension that surfaces replication evidence inline during literature review, a modular Python preprint-notification pipeline (GROBID, DynamoDB, GitHub Actions), and a React/D3.js interactive network visualiser over academic replication data.

Previously a Software Engineer at Netmonastery Inc., shipping production React/Redux frontends, real-time D3.js analytics dashboards, and NLP-driven generative AI features. Before that, Technology Developer at Breeur Solution and Web Developer & Technology Head at Calibrant Classes — delivering client-facing products, role-based dashboards, and payment-integrated platforms from the ground up.

I've been writing code professionally since 2020 across full-stack web apps, data visualisation, ML modelling, and blockchain platforms. Co-author of a 2026 preprint on replication evidence notification (**FLoRA-Notify**) and a published paper on blockchain-based auction systems (IJSREM, 2023).
